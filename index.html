About the project

We don't need to explain how bad the planet is, with global warming, melting ice, floods and forest fires. This project aims to be a hub that brings together different services that will allow people who are involved in making ecological decisions for cities, to be able to help the planet to recover its health a little.

It could contain services that give information about fine particle pollution, if the car drivers have an ecological driving, and even information about noise pollution. Because yes, noise pollution impacts nature.

The first service developed focuses on data from electric vehicle charging spots. This data will be accessible via a dashboard to be able to analyze it, and via a mobile application to help electric car drivers in their search for charging spots.

Dashboard

The dashboard is a tool that gathers the services and contains elements to analyze them. The developed services are independent of the dashboard. They only need to respect some development rules to be attached to the Hub.

Currently, the dashboard already contains analysis tools. Among them, you will find a map to visualize your data, and graphs to observe the evolution of your data. It is the administrator who chooses what type of data should be displayed, and the decision-makers will be able to observe these data.

The list of services can be modified according to the city. Some services may not be interesting for a city.

The administrator has various management tools, including a tool to manage machine learning models for the collected data.

Charging spots Map

The Charging Spots Map application is a mobile application that improves the user experience for electric car drivers. Indeed, the problem that sometimes prevents from choosing an electric car is its battery.

The application is working on this axis. You may not have enough autonomy to make long trips, and you don't know where to find a charging spots. You can spend hours looking for the right charging station, because you want it to be near a place and fast enough to not wait hours for a recharge without doing anything. Then it is possible that when you get to the charging spots, all the outlets are occupied.

To do this, the application uses data retrieved by the charging spots service and a machine learning model to suggest the best charging spots to the driver based on his search parameters.
